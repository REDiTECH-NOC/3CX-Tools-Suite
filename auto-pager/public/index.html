3CX Connection (API)

Used to query queues, active calls, and ring groups via XAPI.

SIP Registration (Audio Playback)

Asterisk registers as a SIP extension with 3CX to make calls and play audio. Use a dedicated extension. Find the Auth ID and SIP port in 3CX Admin → Extensions → Edit → Configuration.

3CX Paging Settings

Paging dial code (e.g. *72) is prepended to the paging extension to make phones auto-answer on speaker. Find it in 3CX Admin → Settings → Advanced → Dial Codes → Paging. Leave blank to call the extension normally.

Relay Agent

The relay agent runs on the PBX and pushes real-time queue data to the auto-pager. When active, it replaces API polling with sub-second updates.
